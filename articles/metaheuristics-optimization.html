Metaheuristics for Optimization: A Deep Dive into Genetic Algorithms

Rosas Behoundja • December 15, 2025 • 10 min read

Table of Contents

    Optimization is at the heart of many computational problems, from machine learning to logistics. When traditional optimization methods fail due to problem complexity, metaheuristics offer powerful alternatives.

    What Are Metaheuristics?

    Metaheuristics are high-level problem-solving strategies that guide the search process to find near-optimal solutions. Unlike exact algorithms that guarantee optimal solutions, metaheuristics trade optimality for computational efficiency.

    The term "metaheuristic" was coined by Fred Glover in 1986. It comes from the Greek words "meta" (beyond) and "heuriskein" (to find).

    Genetic Algorithms

    Genetic algorithms (GAs) are inspired by Darwin's theory of natural selection. They evolve a population of candidate solutions over generations, using mechanisms like selection, crossover, and mutation.

    Key Concepts

    Population
    A set of candidate solutions (individuals/chromosomes)
    Fitness Function
    Measures how good a solution is
    Selection
    Choosing individuals for reproduction based on fitness
    Crossover
    Combining genetic material from parents to create offspring
    Mutation
    Random changes to maintain genetic diversity

    Mathematical Formulation

    Consider an optimization problem:

    $$\min_{x \in S} f(x)$$

    In roulette wheel selection, the probability of selecting individual i is:

    $$P(i) = \frac{f_i}{\sum_{j=1}^{N} f_j}$$

    Python Implementation

    Python 
    import numpy as np
import random
from typing import List, Tuple

class GeneticAlgorithmTSP:
    """Genetic Algorithm for the Traveling Salesman Problem."""

    def __init__(self, cities: np.ndarray, pop_size: int = 100,
                 mutation_rate: float = 0.01, elite_size: int = 10):
        self.cities = cities
        self.n_cities = len(cities)
        self.pop_size = pop_size
        self.mutation_rate = mutation_rate
        self.elite_size = elite_size
        self.distances = self._compute_distances()

    def _compute_distances(self) -> np.ndarray:
        """Compute distance matrix between all cities."""
        n = self.n_cities
        dist = np.zeros((n, n))
        for i in range(n):
            for j in range(i + 1, n):
                d = np.linalg.norm(self.cities[i] - self.cities[j])
                dist[i, j] = dist[j, i] = d
        return dist

    def _fitness(self, route: List[int]) -> float:
        """Fitness = inverse of distance (higher is better)."""
        total = sum(self.distances[route[i], route[i+1]]
                    for i in range(len(route) - 1))
        total += self.distances[route[-1], route[0]]
        return 1 / total

        def _ordered_crossover(self, parent1: List[int],
                           parent2: List[int]) -> List[int]:
        """Ordered Crossover (OX) for permutation encoding."""
        size = len(parent1)
        start, end = sorted(random.sample(range(size), 2))

        child = [None] * size
        child[start:end] = parent1[start:end]

        p2_idx = end
        c_idx = end
        while None in child:
            if parent2[p2_idx % size] not in child:
                child[c_idx % size] = parent2[p2_idx % size]
                c_idx += 1
            p2_idx += 1

        return child

    def _mutate(self, route: List[int]) -> List[int]:
        """Swap mutation: randomly swap two cities."""
        if random.random() < self.mutation_rate:
            i, j = random.sample(range(len(route)), 2)
            route[i], route[j] = route[j], route[i]
        return route
